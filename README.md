# 4-Way Traffic Light Controller

## Overview
This project implements an automatic 4-way traffic light control system using the NE555 Timer and CD4017 Decade Counter in TINA Software.

## Objectives
- Design an automatic traffic signal controller
- Implement sequential signal switching
- Simulate the circuit in TINA Software
- Verify traffic light timing and operation

## Components Used
- NE555 Timer IC
- CD4017 Decade Counter
- LEDs
- Resistors
- Capacitors
- Logic Gates

## Software Used
- TINA Design Suite

## Features
- Automatic signal sequencing
- Four-way intersection control
- Digital logic implementation
- Simulation-based verification
